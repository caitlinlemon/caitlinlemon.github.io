CSS Visual Rules Reference Table

Visual Rule The Purpose Syntax
Font-Family Defines typeface font-family: 'Courier New';
font-size controls size of text font-size: 2em;
font-weight Thick/thin text font-weight:100;
Text-Align Places text L, R, or Center of parent container text-align: left;
Color Color of Text Color: red;
Background-color Color behind text background-color: orange;
Opacity Make element transparent opacity: 0;
background-image set background of element to image background-image: url('https://content.codecademy.com/courses/freelance-1/unit-2/soccer.jpeg');
